# Stripe Price IDs for Beta 2 Cohort System

Generated by: `scripts/setup-stripe-beta2-prices.js`

## Three Pricing Cohorts

1. **Beta 1**: 30% lifetime discount + 1 year free trial (~50 users)
2. **Beta 2**: 20% lifetime discount + 4 months free trial (~100 users)
3. **Public**: Full price + 14 day trial (general public)

## Pricing Details

### Essential Plan (formerly Starter Pro)
- **Beta 1**: €11.20/month, €109.20/year (was €16/month base)
- **Beta 2**: €20/month, €200/year (€25/month base - 20%)
- **Public**: €25/month, €250/year (full price)

### Premium Plan (formerly Professional)
- **Beta 1**: €24.50/month, €310.80/year (was €35/month base)
- **Beta 2**: €39/month, €390/year (€49/month base - 20%)
- **Public**: €49/month, €490/year (full price)

### Agency Plan
- Hidden from public pricing but code remains for special/legacy access
- No Stripe prices created for Agency plan

## Setup Instructions

1. Run the Stripe setup script:
   ```bash
   node scripts/setup-stripe-beta2-prices.js
   ```

2. Copy the outputted price IDs to your `.env` file

3. Run the user migration script (dry-run first):
   ```bash
   node scripts/migrate-users-to-cohorts.js --dry-run
   node scripts/migrate-users-to-cohorts.js
   ```

4. Restart your application to load the new configuration

5. Test subscription checkout for each cohort
